# Curated

**Curated is a no-code platform designed to help users build and manage high-quality software directories and review sites.**

## Our Mission
Curated began as a response to a growing frustration with modern software practices. 

As subscription models became dominant, pricing grew more opaque, cancellation more complex, and vendor lock-in more common. What started as a personal attempt to identify tools that treated users fairly evolved into a structured framework for evaluating software against clear, consistent standards. 

Over time, that framework became the foundation of Curated: a manually reviewed directory built to highlight products that prioritize transparency, reversibility, data ownership, and long-term user respect.

## The Platform
Curated is a manually reviewed directory builder built to highlight products that feature:
* Public and transparent pricing
* No dark patterns
* Frictionless cancellation
* Clear business model
* Respect for user data
